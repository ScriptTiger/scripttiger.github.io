Additional Blacklist Support

Below are assorted versions of Steven Black's unified hosts files reformatted for various other applications for additional support and are synced regularly. Feel free to open an issue if you have any suggestions to support another format, or contact me via the "Contact ScriptTiger" link on the left side of the page.

The FQDN list format is just a standard list of domains, which is the most common blacklist format used by DNSCrtypt, pfSense, etc. If for some reason you need an FQDN list but this one is not working for you, please don't hesitate to open an issue or contact me so we can figure it out and get it resolved for the benefit of yourself and everyone else.

The compressed format is an alternative hosts file format which condenses multiple domain names and comments to a single line. If your are having issues with the regular format hosted by Steven Black or in the Hosts Packages section of this site, you can try this and see if it works for you. Some Windows users suffering from poor performance when using the regular format have noticed dramatic performance gains when using this format. This format may also be better for some mobile applications.

The mcompressed format is the same as the compressed format except it's only halfway compressed to a 5-domain maximum per line instead of the 9 domains per line in the fully compressed format. Some users have found this balance to work better on their systems.
