Geo Tor Table

The Geo Tor Table has been designed to be able to quickly search Tor nodes by a variety of geodata criteria and is updated routinely. While Tor nodes are constantly changing, the Geo Tor Table represents only a snapshot of the Tor nodes present at the time the table is generated. As with other data on this website, updates to the Geo Tor Table are tweeted as soon as they are merged into git, so you can keep up with updates either by following @ScriptTiger on Twitter or by watching the GitHub repo of this website.

Download the Geo Tor Table here!!!

This product includes GeoLite2 data created by MaxMind, available from https://www.maxmind.com.
