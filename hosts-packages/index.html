Unified Hosts Packages

Below are reformatted hosts packages for Steven Black's hosts repo. These hosts files block everything from sidebar ads, annoying YouTube video ads, banner ads, coin miners, and even some invisible embedded analytics trying to gather more information about you. The other files add onto the core Unified Hosts additional blocks.

These hosts packages are updated regularly with Steven Black's hosts files and are reformatted here for the purposes of improving compatibility. The directory structure and naming standards are identical to that of Steven Black's repo to ensure the widest range of compatibility across projects. The ZIP links in this repo are perfect for older legacy hosts file managing software which require hosted archives to pull from as sources. The hosts files in this repo have also been normalized to CRLF to make them more user-friendly for Windows users, while also being just as compatible for any other system, such as Linux, etc.

If you run into any issues using these, you can also try the "compressed" format on the Additional Blacklist Support page.

If you are running a Windows operating system and would like to keep your machine up to date automatically with the latest hosts files from Steven Black, you can also try the Unified Hosts AutoUpdate.
